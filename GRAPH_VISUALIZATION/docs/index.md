# Graph Visualization

欢迎使用 Graph Visualization 项目！这是一个用于图可视化的强大工具，可以帮助你更好地理解和分析图数据。

## 项目简介

Graph Visualization 是一个基于 Python 开发的图可视化工具，具有以下特点：

- 支持大规模图数据的可视化
- 提供多种布局算法
- 交互式操作界面
- 支持自定义样式和主题
- 导出多种格式的图形

## 快速开始

1. 安装项目依赖：
```bash
pip install -r requirements.txt
```

2. 运行主程序：
```bash
python main.py
```
